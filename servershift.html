Platforms: WebGL.
Genre: Puzzle.
PEGI: 3.
Summary: Set up the scenarios before the online players reach the final room by placing elements on the blueprint map as instructed so players can progress, but be careful with speed runners!

Main mechanics:
  • Copy map blueprints: Each map is presented as a 5x5 grid. Place the correct elements in their corresponding spots. Maps are randomly generated each time you play the game.
  • Change player: Switch between players on your server to check for new maps needed. You'll need to manage different players' maps simultaneously, but all on a single grid.
