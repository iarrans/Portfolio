Armagenews

Game Remarks

Armagenews is a puzzle videogame where player takes the role of a journalist who reasearches about fake news for obtaining views by matching title, subtitle and an image. This comes with a twist: if fake news the journalist guesses correctly, lies will become the truth.

The game mixes a 3D scenario with 2D elements and animations. News appearing in game are inspired or copies of real news. This game was made to show that reality surpasses fiction when it comes to disinformation. The game isn't finished.

This project was a submission for the Malagajam Weekend 16 (48 hours jam). It was made on Unity and built as a web game. The game can be played here Armagenews Itch.io
