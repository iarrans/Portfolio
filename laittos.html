Laittos in Crime

Game Remarks


Laittos in Crime is A local multiplayer videogame where 4 players must cooperate to guide a laser from one point to another so they can escape various puzzle scenarios. It's targeted at casual players, so the controls are simple and intuitive, and the levels, which include party videogames' elements, take only a short time to complete.

This project was made as a Thesis for a Game Design Master Degree and passed different playtesting sessions with real players to evolve different gameplay mechanics, adding new ones or discarding not amusing ones.

Most important features in this project are the laser reflection system (optimized with recursive functions) and multiplayer controls management.

Main features and technologies

Game Engine: Unity, C# as main programming language.
Platforms: Windows.
Itch.io page: Laittos in Crime
Github: Laittos in Crime Repository.If not visible, contact me at isabelarrans@gmail.com

Main features:
  • Laser reflection system: Controls which objects can emit a laser, where it can reflect, and which objects the laser collides with. Recursive functions were created for fast processing with Unity's LineRenderer and Raycast libraries.
  • Multiplayer Input Manager: Using Unity's' new input system, assings character and player number to each input conected to the computer by pressing one key. It cand detect both 1 keyboard and 3 controllers or just 4 controllers, so it is also used to control if players inputs are ready for starting the level. For this manager, Player Input Manager and Player Input components are used.
  • Level Management:Loads the available puzzles into the main menu slider, the next level when a level is completed, and manages different level states to adjust the music accordingly.
